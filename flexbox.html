Flexbox.

What is FlexBox?

Flexbox (flexible box) is a layout mode of CSS3.

Why FlexBox?

All this is to use FlexBox in our designs as the best alternative.

For a long time, the only reliable tools compatible with several browsers available to create CSS designs were functions such as floats and positioning. These work, but somehow they are also limiting and frustrating.

The following simple design designs are difficult or impossible to achieve with such tools in any convenient and flexible form:

Center vertically a block of content within your father.Make all the children of a container occupy the same amount of width / high available, regardless of how much width / high it is available. Make all columns in a multi-column design adopt the same height even if they contain a different amount of content.

As you will see in the following sections, FlexBox makes many design tasks much easier. Let's go deeper!

Note: The best alternative is not always the same for each individual. Everything depends on the circumstances and capabilities of the developer.
